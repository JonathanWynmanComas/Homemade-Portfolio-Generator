# Homemade Portfolio Generator

## Description

The **Homemade Portfolio Generator** is a Node.js command-line application that dynamically creates a personalized HTML portfolio page based on user input.

Using prompts in the terminal, users can quickly generate a clean, styled portfolio that includes their name, location, bio, and professional links such as LinkedIn and GitHub.

This project demonstrates the use of Node.js, user input handling, and file system operations to automate web content creation.

---

## Features

* Interactive command-line prompts using Inquirer
* Dynamically generated HTML file
* Built-in CSS styling
* Automatically creates output file in `/dist` folder
* Simple and fast portfolio creation

---

## Technologies Used

* Node.js
* Inquirer (v8.2.4)
* File System (fs)
* JavaScript (ES6 Template Literals)

---

## Installation

1. Clone the repository:

```
git clone https://github.com/your-username/Homemade-Portfolio-Generator.git
```

2. Navigate into the project folder:

```
cd Homemade-Portfolio-Generator
```

3. Install dependencies:

```
npm install
```

---

## Usage

Run the application with:

```
node index.js
```

Then:

* Answer the prompts in the terminal
* Your portfolio will be generated automatically

---

## Output

The generated file will be located at:

```
/dist/index.html
```

Open this file in your browser to view your portfolio.
